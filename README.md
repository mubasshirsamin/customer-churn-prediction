# Telco Customer Churn Prediction
This project analyses and interprents customer churn patterns using Telco's customer churn dataset.
It applies Python, exploratory data analysis, preprocessing, and three machine learning models to
identify the key factors associated with customer churn and predict which factor customer churn
the most.

# Objectives
- Clean and prepare the initial Telco Customer Churn data to make it ready for analysis
- Find the major patterns that drives customer to churn
- Visualise key relationships between churn and customer attributes with the help of several graphs
- Preprocess the data for machine learning
- Train and compare mulitple classification models
- Evaluate model performance using standard classification metrics
- Interpret and present the main factors contributing to customer churn and provide meaningful suggestions
